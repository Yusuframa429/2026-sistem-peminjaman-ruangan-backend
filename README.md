# Sistem Peminjaman Ruangan (Backend)

Backend API untuk sistem peminjaman ruangan kampus, dibangun menggunakan ASP.NET Core.

## Teknologi yang Digunakan
- **Framework:** .NET 8 (ASP.NET Core Web API)
- **Database:** SQL Server Express / LocalDB
- **ORM:** Entity Framework Core
- **Dokumentasi API:** Swagger UI

## Prasyarat Instalasi
Pastikan di komputer anda sudah terinstall:
1. .NET SDK 8.0 atau lebih baru.
2. SQL Server Express (atau LocalDB).
3. Git.
